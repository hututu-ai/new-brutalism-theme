Design Manifesto

🌱 New Brutalism

New Brutalism (新粗野风 / Neo-Brutalism) rejects the polished, invisible UI. Raw structure. Bold borders. Green energy.

⬛

Pure Black Borders

Every panel, card, and widget gets a 2px #000 border. Nothing blends in. Everything declares itself.

🌿

Lime Green Accent

Inspired by the Creator OS dashboard. #84cc16 is our primary — fresh, energetic, unmistakable.

⬜

White on Light Gray

White cards on #f7f7f7 background. Clean, airy, with brutalist structure. Every element is a block.

💛

Yellow Highlights

#fde047 for selections, warnings, and numbers. A warm counterpoint to the green.

🔲

Hard Shadows

3px 3px 0 #000 — no blur, no opacity. Shadows are solid blocks of black. Brutally honest.

🚫

No Subtlety

No gradients. No rounded corners (mostly). No decoration without purpose. Every pixel earns its place.

Get Started

Install in 3 Steps

From zero to brutalist in under a minute.

1

Install the Extension

Search for "New Brutalism Light" in the VS Code extensions panel.

⌘⇧X → Search → Install
2

Activate the Theme

Open the color theme picker and select New Brutalism Light.

⌘K → ⌘T → Select
3

Or Install via CLI

Prefer the terminal? One command and you're done.

code --install-extension new-brutalism-light
💡 Pair with JetBrains Mono or Source Han Sans CN
